# Phonepe_Project
This project is based on the PhonePe Pulse dataset, where I analyzed user growth, transaction trends, and insurance adoption across different states and I also created an interactive dashboard using Streamlit to present the insights clearly.


## 📂 Project Workflow

1. Data Collection  
   - Collected the PhonePe Pulse dataset from GitHub  

2. Data Extraction  
   - Extracted transaction, user, and insurance data  
   - Used SQL queries for required analysis  

3. Data Cleaning  
   - Checked data format  
   - Standardized state and district names  

4. Data Analysis  
   - Analyzed trends in transactions, users, and insurance  
   - Compared state-wise and district-wise performance  

5. Data Visualization  
   - Created charts using Plotly  
   - Used line charts, bar charts, pie charts, and scatter plots  

6. Dashboard Creation  
   - Built an interactive dashboard using Streamlit  
   - Added filters like year, state, and case study  

7. Insight Generation  
   - Identified important patterns and business insights  

8. Suggestions  
   - Gave suggestions based on data findings
